== Installation: ==
	1 .Upload the entire `zigzagblogs` folder to the `/wp-content/themes/` directory
	2. Activate the Theme through the 'Appearance / themes ' menu in WordPress
	
==Frequently Asked Questions:==
	Does this plugin work with newest WP version and also older versions? 	Yes, this plugin works really fine with WordPress 5.1.2. It also works great with WP 5.1
